# 📱 Mobile Sales Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze mobile phone sales data and provide meaningful business insights through dynamic visualizations and key performance indicators (KPIs). This project demonstrates the complete Business Intelligence workflow—from data preparation to dashboard creation—using Microsoft Power BI.

---

## 📖 Project Overview

The **Mobile Sales Dashboard** is designed to help stakeholders monitor sales performance, understand customer purchasing behavior, and identify trends across brands, cities, payment methods, and mobile models.

Using Power BI's data visualization capabilities, the dashboard transforms raw sales data into an easy-to-understand, interactive reporting solution that supports data-driven decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track total transactions and quantity sold.
- Compare sales across different mobile brands and models.
- Identify top-performing cities.
- Analyze customer ratings and payment preferences.
- Monitor sales trends over time.
- Create an interactive dashboard for business users.

## 📂 Dataset Information

The dataset contains mobile sales transactions with the following fields:

| Column Name | Description |
|-------------|-------------|
| Transaction ID | Unique transaction identifier |
| Booking Date | Date of purchase |
| Day Name | Day of the week |
| Brand | Mobile brand |
| Mobile Model | Model of the mobile phone |
| Units Sold | Quantity sold |
| Price Per Unit | Selling price of each unit |
| Customer Name | Customer details |
| Customer Age | Customer age |
| City | Purchase location |
| Payment Method | Mode of payment |
| Customer Ratings | Customer satisfaction rating |
